# 🎓 AI Student Performance Prediction System

A Machine Learning-based web application that predicts student academic performance (Pass/Fail) using Python, Scikit-Learn, and Streamlit.

---

## 🚀 Project Overview

This project analyzes student data and predicts whether a student will pass or fail based on academic and behavioral features such as study time, failures, absences, and previous exam scores.

It also includes an interactive Streamlit dashboard with real-time predictions, data visualization, chatbot, and voice-enabled responses.

---

## 🧠 Features

- 📊 Student performance prediction (Pass/Fail)
- 🤖 AI-powered chatbot (rule-based)
- 🔊 Voice output using gTTS
- 📈 Data visualization (Matplotlib & Seaborn)
- 🎯 Model evaluation (Accuracy, Confusion Matrix)
- 🔐 Login authentication system
- 🖥️ Interactive Streamlit dashboard

---

## 🛠️ Tech Stack

- Python 🐍

- Scikit-Learn
- Matplotlib
